Video Conversion - TEST version

NOTE: Select your input files from a separate directory.

  1. Select an input file from a directory on your computer.
  2. A copy will be made in the directory where this Python script resides. (That working copy is deleted upon completion.)
  3. Then specify an output file. It will be saved in the same directory as the script.
  • WARNING: be careful with naming the output file.
  • In this test version, if a file of that name already exists, the conversion will not occur and an error webpage will be displayed.
  • At this time, only conversion to .mp4 format is supported.
  • Progress will be indicated by the 'loading' animation in the browser tab.
  • Upon success, you will be redirected back to the input page after 5 seconds to do your next conversion.
